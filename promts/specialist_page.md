## Создание страницы специалиста 

Необходимо сверстать и стилизовать страницу специалиста для данного сайта. При работе опирайся на следующие инструкции:

- Верстку размести в файле resources\views\specialists\page.blade.php
- Верстку размещай в указанном файле между элементами @section('main') ... @endsection
- Стили в формате scss в файле public\scss\_personal-page.scss
- В начале файла со стилями должно быть подключение файла с переменными, реализованное следующим образом: @use 'vars' as *;. 
- При создании стилей используй переменные которые описанны в файле public\scss_vars.scss. Не используй другие имена переменных! Используй только те которые есть в указанном файле.


### Верстка страницы

В начале вставь следующие элементы:

```php

<x-breadcrumbs.main :title="$title"></x-breadcrumbs.main>
<h1>{{ $specialist->title }}}</h1>
<p>{{ $specialist->position}}<p>

```
Далее должен идти блок состоящий из 2 колонок:

- левая колонка содержит фото специалиста, фото с соотношениес сторон 9:16. src фото подставь из $specialist->photo, аттрибут alt заполни из $specialist->title

- правая колонка занимает все свободное прострнаство в нее выводится в эту секцию должно выводится описание из параметра $specialist->description и сертификаты из json параметра $specialist->certificates. Сертификаты выводятся в виде малых превю, каждый элемент оберни в ссылку с href="#"
