# Создание структуры БД

Для каждой перечисленной в этом разделе сущьности нужно создать:
- Модель
- Миграцию
- Сидер (Сидер создается пустой, к нему подключается фасад DB и Str)
- Контроллер (если в описании сущьности будет указание контроллер не создаем)

При наличии в описании полей сущьности поля Слаг/slug нужно создать мутатор который делает это поле из поля заголовка или имени.

## Сущьность - Сертификаты

**Необходимые поля:**
- Наименование ( строка, nullable)
- Порядок вывода (Целое число, по умолчанию 0)
- Файл (строка, 700 символов)
