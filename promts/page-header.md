# Создание шапки простой страницы

Изучи подходы к верстке в данном проекте и создай мне blade компонет page-header который будет лежать в директории resources\views\components\headers.

Стили для данного компонента положи в файл public\scss\_page-header.scss в начале которого подключи файл с переменными @use 'vars' as *;

## Описание блока
- У блока на заднем фоне изображение <> которое заблюрено css фильтром 
- Края блока закруглены скругление взять из переменной $border_rad_big
- в блоке расположен заголовок h1 с текстом "Контакты"
- Над заголовком расположи хлебные крошки - <x-breadcrumbs.main title="Контакты"></x-breadcrumbs.main>
